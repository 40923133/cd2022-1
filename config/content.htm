建立倉儲

先進去老師倉儲的cmsimde site，複製網址

到Create a new repository點開import a repository

貼上剛剛複製的網址，Repository Name設為cd2022

接著到小黑窗

cd tmp

git clone --recurse-submodules https://github.com/a40923101/cd2022.git

cd cd2022

cms

複製網址 https://127.0.0.1:9443/ 就可以開啟倉儲了。

tokens

到developer setting 中創建一個沒有期限的token並且將下面選項都打勾之後會出現一串文字碼

開啟config到白窗，複製貼上指令然後儲存。接著到小黑窗acp，檢查完成。
